ใบสมัครงาน AIKM
AI Knowledge Manager — รู้ทุกฝ่าย

ตำแหน่งอื่นๆ: ← AI Employee 🧠 AI Knowledge Manager

AIKM ต่างจาก AI Employee ตรงที่รู้ข้อมูลทุกฝ่าย วิเคราะห์ข้ามฝ่าย และมีบุคลิกที่เลือกได้ · ติ๊กให้มากที่สุด กด Copy แล้วแปะใน Instruction ทันที

1. ตัวตนและตำแหน่ง

กรอกสั้น ๆ ให้รู้ว่า AIKM ตัวนี้เป็นใคร

มุมมองผู้เชี่ยวชาญเสริม เลือก 0-2 ข้อ

2. JD: หน้าที่หลัก

เลือกได้หลายข้อ หรือพิมพ์เพิ่มเองด้านล่าง

3. Personality ⭐

เลือก 1 แบบที่ตรงกับทีมของคุณที่สุด

4. รูปแบบคำตอบ

ให้ AIKM ตอบออกมาเป็นรูปแบบที่ทีมใช้ต่อได้ทันที

5. เขตแดนแบบ SoD

แบ่งเขตให้ชัด: AIKM รู้ข้อมูลข้ามฝ่ายแต่ต้องมีขอบเขตที่ชัดเจน

ควรรู้ / Access

ห้ามรู้ / Restricted

6. Human Checkpoint

เรื่องไหนต้องให้คนตรวจ ก่อน AIKM รายงานออกไป

7. คำถามสัมภาษณ์ AIKM

ใช้ถามหลังตั้งค่า เพื่อทดสอบว่า AIKM พร้อมทำงานไหม

8. Skill เสริม 🆕

เลือกได้ 1 อัน ตามบุคลิกที่เลือกไว้ในหมวด 3

🟡 Cross Dept Dashboard (H5-S4)
รวบรวมข้อมูลจากฝ่ายขาย+จัดซื้อผ่าน Template มาตรฐาน สร้าง HTML Dashboard แสดงยอดขาย/ค่าใช้จ่ายข้ามฝ่าย เหมาะกับ Analyst ที่ต้องการเห็นภาพรวมเป็นตัวเลขในทันที
🟢 Business Coach (H7-S3)
ให้คำปรึกษาแบบโค้ช ถามก่อนแนะนำ ไม่บังคับทิศทาง เสนอทางเลือก 3 ระดับ (ทำได้เลย / วางแผน / ระยะยาว) เหมาะกับ Advisor ที่ต้องการชี้แนะโดยไม่ก้าวก่าย
🔴 Problem Analyzer (H7-S1)
วิเคราะห์ปัญหาด้วย Root Cause (5 Whys) พร้อมแนวทางแก้ 3 ระดับ เพิ่มเติม: SWOT, Business Model Canvas เหมาะกับ Investigator ที่ต้องการเจาะลึกต้นตอของปัญหาข้ามฝ่าย

Guideline — AIKM

จุดที่ต้องจำตอนสร้าง AI Knowledge Manager

  • JD
    รู้ทุกฝ่าย วิเคราะห์ข้ามฝ่าย Escalate ตามตาราง ไม่เดาเอง
  • Personality
    เลือก 1 ใน 3 — Advisor / Analyst / Investigator — กำหนดน้ำเสียงและวิธีรายงาน
  • Access
    รู้ข้อมูลข้ามฝ่ายเท่าที่จำเป็นต่อการวิเคราะห์และ Escalate
  • Restricted
    ข้อมูลส่วนบุคคล เงินเดือน รหัสผ่าน และ ห้ามตัดสินใจแทนผู้จัดการเด็ดขาด
  • Human Checkpoint
    ก่อน Escalate และก่อนสรุปว่าระบบขาดตรงไหน ต้องให้คนยืนยัน
